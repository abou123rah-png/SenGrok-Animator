# Guide de Distribution Gratuite (GitHub) 🚀

Pour rendre votre extension **SenGrok Animator** disponible partout et gratuitement, la meilleure solution est **GitHub**. Voici les étapes à suivre :

## 1. Créer un Compte GitHub (Gratuit)
- Allez sur [github.com](https://github.com) et créez un compte.
- GitHub est le standard mondial pour le code. C'est gratuit et les pages GitHub sont très bien référencées sur Google.

## 2. Créer un Repository (Dépôt)
- Cliquez sur le bouton "New" (Nouveau dépôt).
- Nommez le dépôt : `SenGrok-Animator`.
- Mettez-le en **Public** pour que tout le monde puisse le voir.
- Ne pas ajouter de README ou de Licence (nous les avons déjà dans le dossier).

## 3. Téléverser votre Code
- Dans votre nouveau dépôt sur GitHub, cliquez sur "uploading an existing file".
- Faites glisser tous les fichiers de votre dossier `grok_automation` dans la fenêtre.
- Cliquez sur "Commit changes".

## 4. Visibilité et Recherche (SEO)
- Retournez sur la page principale de votre dépôt (cliquez sur le nom du projet en haut à gauche).
- Dans la colonne de droite, cherchez la section **"About"**.
- Cliquez sur l'icône **⚙️ (petit écrou)** à droite de "About".
- Dans la fenêtre qui s'ouvre :
  - **Description** : *"Automatisation IA Vidéo pour Grok par ciis-doro. Batch prompts et scripts d'animation (Sénégal)."*
  - **Topics** : Ajoutez `grok-ai`, `automation`, `senegal`, `chrome-extension`, `artificial-intelligence`.
- Cliquez sur **"Save changes"**.

## 5. Partager
- Partagez le lien de votre dépôt sur les réseaux sociaux (X, LinkedIn, Facebook).
- Les utilisateurs n'ont qu'à télécharger le ZIP, le décompresser et l'ajouter à Chrome via le "Mode Développeur" (comme expliqué dans le README).

Avec cette méthode, vous ne payez rien (0 FCFA) et votre travail est professionnellement exposé au monde entier !
