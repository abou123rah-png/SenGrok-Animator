# 🇸🇳 SenGrok Animator - L'Automatisation d'IA Vidéo par ciis-doro

**SenGrok Animator** est une extension Chrome gratuite et performante conçue pour automatiser la création de vidéos et d'images sur [xAI Grok](https://grok.com/imagine). Optimisée par **ciis-doro**, elle offre une interface fluide (Sky Blue Senegal) et des fonctionnalités uniques pour les créateurs de contenu.

## 🚀 Fonctionnalités Clés
- **Batch Video Generation** : Lancez plusieurs prompts en une seule fois.
- **Content Builder** : Gérez vos prompts visuels et vos dialogues/scripts de manière coordonnée.
- **Modes Multiples** : Text-to-Video, Frame-to-Video, Text-to-Image.
- **Smart Queue** : File d'attente intelligente avec gestion des délais pour éviter les blocages.
- **Protection Grok** : Interface sécurisée qui s'active uniquement sur la page Grok Imagine.
- **Guide Intégré** : Instructions directement accessibles dans l'extension.

## 📦 Comment Installer (Gratuitement)
Comme cette extension est en développement open-source, voici comment l'installer :
1. Téléchargez le code source (Zip) depuis le dépôt officiel : [GitHub SenGrok Animator](https://github.com/abou123rah-png/SenGrok-Animator)
2. Décompressez le fichier sur votre ordinateur.
3. Ouvrez Chrome et allez dans `chrome://extensions/`.
4. Activez le **Mode développeur** (en haut à droite).
5. Cliquez sur **Charger l'extension non empaquetée** et sélectionnez le dossier décompressé.
