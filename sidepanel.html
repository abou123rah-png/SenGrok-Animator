Guide d'utilisation SenGrok

🚀 Commencer

Utilisez le Content Builder pour créer votre séquence d'animations.

📝 Modes de Génération

  • Text to Video: Créez une vidéo à partir d'une description.
  • Frame to Video: Animez une image existante.
  • Text/Image to Image: Générez des visuels statiques.

💬 Dialogues & Scripts

Pour les animations complexes, utilisez la section Dialogues. Vous pouvez y mettre le texte qui sera coordonné avec chaque scène. Séparez vos scènes par une ligne vide dans le prompt principal.

⚡ Paramètres Avancés

  • Concurrent Prompts: Nombre de tâches lancées en même temps.
  • Prompt Delay: Nombre de secondes d'attente entre chaque requête.

▶️ Exécution

Cliquez sur Add to Sequence pour remplir la file d'attente, puis sur Start Batch pour lancer l'automatisation.
